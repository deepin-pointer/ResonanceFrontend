# ResonanceFrontend
雷索纳斯数据统计前端
Frontend for statistics collection of Resonance Solitice

基于vue3 + ts + axios + pinia + element plus, 施工ing

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
